## 因缘际会

我在高中二年级的某个月假（我们学校每月放假2天）的下午，在某个旧书店闲逛，在一堆实在卖不出去，胡乱摆放着的书堆里翻出来一本书。我站在店里看了很久，直到天黑。大概是把它粗略的看完了，这本书介绍了一些21世纪紧缺的金领职业和相关内容。那时候我第一次听说金领这个词，我顺利把它和我以前听说过的蓝领和白领这两个词联系了起来。

“喔，牛逼，还有这个称呼。”

大部分职业我已经忘记了，但是我对股票交易员留下了深刻的印象。那之后我就开始关注相关的信息，高三的时候, 我有了一台手机 `Nokia 6120c` 安装了电子书软件，下载一些股票证券相关的电子书，我囫囵吞枣地看了不少，内容自然也是全忘了，记住了一些词汇，蜡烛图，长线，短线，价值投资，基本面分析...

大一的时候，我选修了证券投资的辅修课，这种课也只有大一的时候可以选修，我是进了课堂才知道这种课程纯粹是混学分的，“我当初可是因为敢兴趣去选修的啊“。

不过，我并没有太大的抱怨，这个老师讲课非常有意思。我还记得我们的老师叫罗森，他会在课堂上打开电脑投屏，展示自己在玩一些棋牌和德州扑克，会说自己打牌和在线玩棋牌赢了多少钱，他会强调自己不是赌博，可是以我大一的心智和知识水平怎么可能理解的了呢。

他在课上介绍了一本书，叫《股票作手回忆录》，讲了作者的故事，作者极具戏剧性的经历深深吸引了我，“炒股票炒到交易所禁止他入内”。不过由于罗老师的普通话比较差，我听不清楚作者的名字，但是书名本身已经很有趣了，“作手” 这个词汇让我印象深刻，课后我就买到了这本书，原来作者的名字叫利夫莫尔，我当时记的是 _Live more_ 觉得这个名字也很牛逼，居然是个短句子，以至于怀疑这是个类似网名或者笔名的东西。多年后我才知道他的名字是 _Jesse Lauriston Livermore_

_Liver more_, "肝脏，更多"，可能是个热爱吃猪肝的人吧。

后来到了大四的时候，我所在的大学里举办了一场长江证券冠名的大学生模拟炒股比赛，我在 10 天的疯狂操作中，盈利 9%，勇夺桂冠。

这一切都是我的实力的部分体现...

前面两周9个交易日中，我通过不断地分析 macd 和 k线 进行操作，一边百度一边看线，不停的买进卖出，每天收盘还坚持看盘，试图找到民间传说的盘感，类似于 _Livermore_ 因为每天抄价格公告板而获得的能预测价格的某种能力。

多天的操作下来我已经成功地亏损了3-4%，我想我肯定没戏了。但是最后一天的早上，我把所持有的各支股票全部卖出，随后在列表里选择股票，我All in了一个名字看上去较厉害的股票--神州泰岳。同时我还搜索了一下，他们在开发一款叫做飞信的软件，我前不久在诺基亚上还安装过，点开看了下就关掉了，前些天还经常看到他们的新闻，似乎很不错。

这可能是我仅有的基本面分析吧，基本面分析，这是个专业术语。

然后我就爬上床去睡午觉，入睡前想着这个比赛就算过去了，没什么念想了。但是午睡起来，发现这货居然涨停板了，到收盘我才敢确定我居然扭亏为盈了。

后来我坐公交车亲戚去长江证券的最近的营业部领奖，证券的蔡经理把所有参赛账户的盈利情况给我看，几百人参赛，只有30人多的账户收益是大于0的，这些30多有收益的人中差不多20人是开户之后没有过任何操作，本金丝毫未动的。排前面的9位是操作之后还盈利了的，说到这里，诸位看官自己品。

不才，我就是第一名。

领奖的时候，我认识了第二名的学长，他的累积收益是2%，他来自石油工程学院，他站我旁边合影，他悄悄在我耳边跟我说他注册了几个小号都没干过我。我当时惊呆了，“居然还有这种操作”。

后来大四，我因为我哥的关系和安排去到新疆跑市场，过程中联系了这个学长，因为他大我一届，所以他已经考到西南财大就读研究生一年级了。

在他暑假的时候，我辞了职去到了成都的西南财经大学，经过他的安排，为住在他的宿舍里备考证券投资专业的专业硕士项目。在那里我看了几本宏观经济学的书，学起来饶有兴致，但是课本我看不进去，备考毫无头绪。

由于有部分研究生暑假也在学校住校，我对研究生的生活和学习状态有了一个亲身的体验。过程中我听说了学长对自身这种跨专业来的专业硕士未来就业的担忧，加上迫于手上没钱了，我于是放弃了这条路去找工作了。

一晃这都是十年前的事情了。

前不久，我签了一家国内的外包公司，他们把我外包到广州黄埔一家外企，这家外企把我外包到大楼隔壁的另一家外企。

这家国内的外包公司名字叫做神舟数码，我简单查了一下，原来就是当年涨停板的神舟泰岳的子公司，真是缘分。

Anyway

这些只是我简单介绍了一下我和交易投资相关的细微的缘分。

## 金融产品术语表

| 中文名称 | 英文名称        | 说明                                                                                                                                                                           |
| -------- | --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 股票/ETF | Stocks/ETFs     | 股票是公司发行的股份，代表对公司的所有权；ETF（交易所交易基金）是一种在交易所上市交易的基金，通常跟踪特定指数的表现。投资者可以像买卖股票一样买卖ETF，提供了多样化投资的机会。 |
| 期权     | Options         | 期权是一种金融合约，赋予持有者在特定时间以特定价格买入或卖出标的资产的权利，但没有义务。期权分为看涨期权（Call）和看跌期权（Put），用于对冲风险或投机。                        |
| 期货     | Futures         | 期货合约是一种协议，要求在未来某个时间以约定价格买入或卖出某种资产。期货合约通常用于商品、金融工具等的交易，具有较高的杠杆效应。                                               |
| 现货货币 | Spot Currencies | 现货货币交易是指在外汇市场上立即交割的货币交易，通常是在交易日内完成。现货市场价格反映了当前市场供需关系。                                                                     |
| 债券     | Bonds           | 债券是借款合同，投资者向发行者（如公司或政府）提供资金，并在未来某个日期收回本金及利息。债券通常被视为较低风险的投资工具。                                                     |

### 举例子：IBKR 特斯拉股票交易佣金计算 <Twemoji emoji="clinking-beer-mugs" />

如果您在盈透证券（Interactive Brokers）购买特斯拉股票，佣金将按照美国<Twemoji emoji="flag-us" />市场的佣金结构进行计算。以下是基于特斯拉股票价格为400美元的佣金计算总结。

⚠️ 注意：

- 在美国市场，IBKR Lite账户享有零佣金交易，但仅适用于美国和印度的散户投资者。
- IBKR Pro账户则适用于全球用户，提供更灵活的佣金选择。

**佣金结构**

| 佣金类型     | 佣金标准    | 最低佣金 | 最高佣金      |
| ------------ | ----------- | -------- | ------------- |
| **分层定价** | 每股 $0.005 | $1.00    | 交易价值的 1% |
| **固定定价** | 每股 $0.005 | $1.00    | 交易价值的 1% |

**交易示例**

- **股票**: 特斯拉 (Tesla)
- **股票价格**: $400
- **购买数量**: 10 股

| 计算方式     | 计算过程                            | 最终佣金              |
| ------------ | ----------------------------------- | --------------------- |
| **分层定价** | (10 股 x 0.005 美元/股 = 0.05 美元) | $1.00（最低佣金适用） |
| **固定定价** | (10 股 x 0.005 美元/股 = 0.05 美元) | $1.00（最低佣金适用） |

### 总结，与小散户无关<Twemoji emoji="clinking-beer-mugs" />

- 无论选择分层定价还是固定定价，最终收取的佣金均为 **$1.00**，因为计算出的佣金低于最低佣金要求。
- 如果购买数量增加到200股：
  - **分层定价**: (200 股 x 0.005 美元/股 = 1.00 美元)，最终佣金为 $1.00。
  - **固定定价**: (200 股x 0.005 美元/股 = 1.00 美元)，最终佣金为 $1.00。

## 相关问题

### 每次能买卖多少股的单位数量？

#### 不同市场的最小交易单位

| 市场类型 | 最小交易单位              | 说明                                                                                          |
| -------- | ------------------------- | --------------------------------------------------------------------------------------------- |
| 港股     | 1手                       | 每手所含的股票数量根据股票价格决定，可能是100股、500股、1000股等。例如，腾讯控股每手为100股。 |
| A股      | 100股                     | 在中国上海证券交易所和深圳证券交易所，最小交易单位为100股或其整数倍。                         |
| 美股     | 1股                       | 在美国市场，最小的交易单位是一股，可以进行任意数量的买卖。                                    |
| 债券     | 1000元面额为一手          | 债券以人民币1000元面额为一手进行交易。                                                        |
| 证券回购 | 1000元标准券或综合券为1手 | 证券回购以1000元标准券或综合券为1手进行申报。                                                 |

**示例说明**

- **港股示例**:
  - 腾讯控股每手为100股，如果投资者想要买入300股，则需要买3手（300/100）。
- **A股示例**:

  - 如果某个A股投资者想买入250股，则需要买3手（300股）并持有50股碎股。

- **美股示例**:

  - 在美股市场，投资者可以选择购买1股、5股、10股等，没有最小交易单位限制。

- **债券示例**:
  - 债券交易以人民币1000元面额为一手，因此投资者至少需要购买1000元面额的债券。

| -------------------- | --------------------------------------- | ---------------------------------------------------------------------- |
